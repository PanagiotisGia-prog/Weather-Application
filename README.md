# Weather-Application
A simple console-based weather forecast simulator.

## Description
Weather Simulator is a lightweight application that generates random weather forecasts for a specified number of days. It simulates realistic weather conditions by adjusting precipitation types based on temperature (rain becomes snow below freezing point).

## Features
- Prompts for the number of days to forecast with error handling
- Generates random temperatures between -5째C and 40째C
- Randomly selects from Sunny, Cloudy, Rainy, and Snowy conditions
- Automatically converts rain to snow below 0째C and snow to rain above 0째C
- Displays:
  - Average temperature
  - Minimum temperature
  - Maximum temperature
  - Most common weather condition

## Requirements
- .NET 8 or later
- C# 12.0 or later

## Usage
1. Run the application
2. Enter the number of days for the weather forecast
3. View the generated forecast with temperatures and weather conditions
4. Review the weather statistics summary
